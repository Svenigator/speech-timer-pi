📖 OSC-Adressen-Referenz

Eingehende Befehle (an den Timer)

AdresseParameterAktion
/timer/start—Timer starten
/timer/pause—Pausieren / Fortsetzen
/timer/stop—Stoppen
/timer/reset—Zurücksetzen
/timer/presetintPreset nach ID starten (z.B. 1, 2, 3)
/timer/preset/namestringPreset nach Name starten
/timer/durationintDauer in Sekunden setzen & starten
/timer/adjustintZeit anpassen (±Sekunden)
/display/modestring"timer" oder "clock"
/display/mode/toggle—Zwischen Timer/Clock toggeln

Ausgehende Nachrichten (vom Timer)

AdresseTypBedeutung
/timer/state/runningint1 = läuft, 0 = läuft nicht
/timer/state/pausedint1 = pausiert, 0 = nicht
/timer/state/phasestringidle / normal / warning1 / warning2 / overtime
/timer/state/remainingintVerbleibende Sekunden (negativ = overtime)
/timer/state/elapsedintAbgelaufene Sekunden
/timer/state/durationintGesamtdauer in Sekunden
/timer/state/time/formattedstringFormatierte Zeit „MM:SS" o. „-MM:SS"
/timer/state/presetstringAktueller Preset-Name
/timer/state/overtimeint1 = Zeit überschritten
